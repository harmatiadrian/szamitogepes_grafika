# Számítógépes Grafika
*Ez a projekt a megosztás egyszerűsítése miatt publikus, és nem biztos hogy valaha is elkészül!<br>
A szövegben az ismertebb referenciák a Google keresőmotor miatt torzítottak*

A játék egy 3D-s FPS játék lesz ami a Primeval világából merít ihletet.
- A játékban a játékos fő célja a sorozatból jól ismert anomália bezárása/biztosítása és az esetleges átjött állatok elaltatása, és természetesen a túlélés.
- Későbbiekben akár több pályával egy Phasmophobia stílusú játékká fejlődhet, ennek alapjait akarom megalapozni. 
- A karakter opcionálisan teljes realizmus követése érdekében sérülés után vérzést kaphat, továbbá néhány állat mérgezhet is.
- Az állatok között sebességkülönbség, inteligencia/taktika különbséggel *(ha lessz többfajta állat)*.

## Opcionális
- Az elaltatott állat az anomálián át való visszaküldése 
- teljes vércsoport szimuláció implementálása
